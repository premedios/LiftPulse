# Quickstart: App Navigation

**Feature**: App Navigation  
**Date**: 2026-02-14  

## Overview

The app navigation provides a bottom navigation bar with 4 tabs: Home, Workouts, Trends, and Profile. Tap any tab to switch screens instantly.

## Getting Started

1. **Launch the app**: The app opens to the Home tab by default.
2. **Navigate**: Tap the bottom navigation icons to switch between tabs.
3. **Home tab**: Displays centered "Home" text with no app bar.
4. **Other tabs**: Show an app bar with the tab name (Workouts, Trends, Profile).

## Features

- **Responsive design**: Works on iOS and Android with Material 3 styling.
- **Smooth transitions**: 60fps animations between tabs.
- **Accessibility**: Screen reader support and scalable text.
- **Performance**: Efficient rendering with no memory leaks.

## Troubleshooting

- If navigation feels slow, ensure you're on a device with 60fps capability.
- For accessibility issues, check that screen readers announce tab changes.
- All tabs should load instantly as they contain no heavy content yet.
